## Description
Koa 2.x API using:

 - Koa 2.x
 - Mongodb
 - Mongoose
 - Asynchronous Functions (Async/Await)

## Running
Install dependencies
```
npm install
```

Start a Local Server
```
npm start
```

Run Test
```
npm test
```

**Note : Please make sure your MongoDB is running before using ```npm start``` **
